# 🎭 Jokes Viewer Application

A sleek, modern React application that fetches and displays random jokes from a public API. This project is excellent for beginners who want to learn how to integrate APIs into a React frontend while styling their application with modern CSS techniques.



## 🎓 What Students Will Learn

By following or reviewing this project, students will discover how to:

1. **React State & Props (`useState`)**
   - Manage application state to handle API responses gracefully.
   - Handle loading or `null` states before API data is available.

2. **Side Effects & Data Fetching (`useEffect`)**
   - Use `fetch()` with `async/await` to call external APIs (FreeAPI.app).
   - Trigger initial API calls when the component mounts.

3. **Modern UI/UX & CSS Styling**
   - Build a world-class, premium dark mode aesthetic (Vercel/NextJS style).
   - Implement Glassmorphism effects using `backdrop-filter`.
   - Work with linear gradients for backgrounds, text (`background-clip`), and buttons.
   - Create smooth hover interactions and CSS transitions.

4. **Event Handling**
   - Trigger new data fetches with simple button `onClick` events.

## 🛠️ Tech Stack
- **React 18+** (with Vite)
- **CSS3** (Modern Web Design, Flexbox, Gradients)
- **FreeAPI** (Public Jokes API)

## 🏃‍♂️ Getting Started

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation & Run

1. Clone this repository or download the source code.
2. Navigate to the project folder:
   ```bash
   cd Jokes-Viewer-Application
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open up your browser and navigate to the provided localhost URL (e.g. `http://localhost:5173`).

Enjoy the jokes! 😄
